# Microservicio Películas

## Descripción
Microservicio desarrollado en Java con Spring Boot para gestionar películas.

## Funcionalidades
- Crear película
- Listar películas
- Buscar película
- Actualizar película
- Eliminar película

## Estructura
- Controller
- Service
- Model

## Tecnologías
- Java 25
- Spring Boot
- Maven

## Cómo ejecutar
```bash
mvn spring-boot:run
